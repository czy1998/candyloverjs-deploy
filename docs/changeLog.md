# 更新日志

`@candyloverjs/deploy` 从 1.0.0 开始 [严格遵守 Semantic Versioning 2.0.0](https://semver.org/lang/zh-CN/) 语义化版本规范

## 发布周期

- 修订版本号： 日常 bugfix 更新（如果有紧急 bugfix，则任何时候都可发布）
- 次版本号：不定期发布一个带有新特性的向下兼容的版本
- 主版本号：含破坏性更新和新特性，不在发布周期内
